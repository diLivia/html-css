Principais formatações

Negrito/Destaque

Nessa frase temos um termo em negrito usando a tag B (não semântica).

Nessa frase temos um termo em negrito usando a tag STRONG (semântica).

Itálico/Ênfase

Nessa frase temos um termo em itálico usando a tag I (não semântica).

Nessa frase temos um termo em ênfase usando a tag EM (semântica).

Texto marcado

Podemos criar também um texto marcado usando a tag MARK.

E no outro parágrafo, temos outro texto marcado.

Texto grande e pequeno

Estamos criando o texto grande e um texto pequeno nesse parágrafo (BIG é não semântica)(SMALL é semântica).

Texto deletado

Podemos marcar um texto como excluído, para indicar que ele deve ser lido, mas não considerado, com a tag DEL.

Texto inserido

Podemos marcar um texto inserido para dar uma ênfase e indicar que ele foi adicionado depois, com a tag INS.

Existe também o sublinhado com a tag U (não semântica).

Texto sobrescrito

Para inserir coisas do tipo x20+3, usamos a tag SUP.

Texto subscrito

Para inserir coisas do tipo H2O, usamos a tag SUB.
